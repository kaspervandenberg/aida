Welcome to the AIDA Search Interface

This interface may be used to query indexes generated using AIDA, possibly using concepts retrieved from a thesaurus.

To begin, select an index on the top-left of the page and possibly a field ("content" usually suffices). You can then type in any query to search that index. Note that it assumes a Boolean OR by default. To get a Boolean AND, prefix your terms with a '+' character.

Moreover, you can also select a server and SKOS repository on the bottom right. As soon as the thesaurus is loaded, you can drag concepts onto the "My Query" node in the Query Builder at the top right. These concepts will then be searched for, in conjunction with a typed query (if any).

You can right click on the added nodes in the Query Builder for more advanced search options.

Comments, questions, and other remarks can be sent to aidadev@lists.vl-e.nl.
