Where to Send a Bug Report

E-mail Bug Reports

Please report bugs via email to:

  • bugs@alias-i.com

Newsgroup Bug Reports

Once we have recreated your bug, we will post news about it on the newsgroup and get back to you about it.

  • LingPipe Newsgroup

How to Write a Bug Report

The purpose of a bug report is to allow us to fix the bug. To do that, we need to be able to recreate it. To recreate a bug, your bug report should be as specific as possible.

How you can Help Us

Please send us as much of the following information as possible:

  • Hardware Platform (e.g. AMD64, Sparc)
  • Operating System (e.g. Windows, Linux)
  • Java Version (e.g. Java 1.4.2)
  • LingPipe Version (e.g. LingPipe 3.4.0)

It's even more helpful if you can provide:

  • Description of what you expected to happen
  • Description of what actually happened, ideally in the form of system output and a stack trace
  • Description of what steps we can take to recreate the bug, ideally in the form of a small sample program that demonstrates the bug.
