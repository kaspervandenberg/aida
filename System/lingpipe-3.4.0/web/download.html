Getting Started

The first two resources for getting started are:

  • Installing LingPipe

Instructions for installing Java and running the out-of-the-box demos. Optional information on installing Ant (for builds), JUnit (for tests) and necessary Windows utilities for compression and command-line operation.

  • API Tutorials

Tutorials covering the LingPipe API and several application domains.

The core LingPipe API and models are distributed as a single file, which contains everything you need to get started:

  • Precompiled LingPipe jar file
  • GUI and command-line demos
  • Full source code with unit tests
  • Precompiled JavaDoc API documentation
  • Extensive tutorial examples
  • Third-party libraries [see third-party licenses]
  • A complete copy of this web site
  • Sample models for the demos and tutorials

Many of these items are also available online. See the navigation bar on the left side of this page for links.
